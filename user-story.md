## User Stories

### 1. Create Account
As a customer  
I need to create an account  
So that I can store my details  

### 2. Read Account
As a customer  
I need to view my account  
So that I can check my information  

### 3. Update Account
As a customer  
I need to update my account  
So that I can modify my details  

### 4. Delete Account
As a customer  
I need to delete my account  
So that I can remove my data
